# 🛒 OneTapp AI Retail Analytics Assistant

An AI-powered Retail Analytics Assistant that enables business users to ask natural language questions about sales, inventory, promotions, pricing, and regional performance.

## Features

- Natural language business queries
- AI-generated SQL using Groq LLM
- SQLite database querying
- Business insight generation
- Streamlit interface
- SQL error handling
- Retail domain validation

## Tech Stack

- Python
- Streamlit
- SQLite
- Groq LLM
- Pandas

## Example Questions

- Which region sold the highest units?
- Which category has the highest inventory?
- Show average price by category.
- Which season has the highest demand forecast?

## Run Locally

```bash
pip install -r requirements.txt

streamlit run app.py
```
